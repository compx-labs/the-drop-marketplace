# The Drop Marketplace - Frontend

Frontend application for The Drop Marketplace built with Vite, React, and Tailwind CSS v3.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- **Vite** - Build tool and dev server
- **React** - UI library
- **Tailwind CSS v3** - Utility-first CSS framework
- **Fonts** - JetBrains Mono, Space Mono, Fira Code
